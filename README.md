# Dota2名主播头像

## 简介

这个是一个Dota2名主播头像的合集。

## 环境依赖

```bash
[sudo] apt install -y imagemagick
```

## 生成头像

```bash
bash resize.sh
```

## 生成文档

```bash
bash create_readme.sh
```
